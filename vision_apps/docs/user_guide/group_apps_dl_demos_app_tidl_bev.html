Fast Bird Eye View Application

Introduction

This application shows a demo of Fast-BEV 3D application which runs Fast-BEV deep learning algorithm on multiple input channels such as front, right, back, front-right, front-left and left cameras. The demo takes as input a sequence of NV12 images of resolutions 1920x698 pixels coming from different input channels. The input is resized using a multi-scalar hardware (MSC) to a 704x256 resolution which is provided to a Fast BEV 3D object-detection network for doing object detection. All the inputs are pre-processed on A72 into 4D Tensor before providing to TIDL running on C7x DSP. The output of TIDL is then post-processed on A72 which draws 3D boxes around objects in image space. Post processing also draws boxes around objects in the top-view of 6-camera fusion space. Lidar point cloud is loaded separately into post processing as a golden reference for the drawn boxes.

Steps to run the application on J7 EVM (Linux + RTOS mode)

  1. Build the application and related libraries as mentioned in Build Instructions for Linux+RTOS mode
  2. A sample config file "app_bev.cfg" is provided under "/opt/vision_apps/" folder on the rootfs partition.
  3. Edit config file to modify the fields as required
  4. Run the app as shown below
    cd /opt/vision_apps
    source ./vision_apps_init.sh
    ./run_app_tidl_bev.sh
