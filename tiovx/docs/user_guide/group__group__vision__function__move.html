vx_reference  first,
vx_reference  second

Move data from one object to another. Same as Swap but second parameter is an output.

Note
An implementation may optimize away the move when virtual data objects are used.
Parameters
[in]graphThe reference to the graph.
[in,out]firstThe first data object.
[out]secondThe second data object with meta-data identical to the input data object.
Returns
vx_node.
Return values
vx_nodeA node reference. Any possible errors preventing a successful creation should be checked using vxGetStatus
