Introduction

The Heterogeneous App is a test application used to confirm the heterogeneous use case.

The Heterogeneous App is not meant for production code. It is only used for testing and verifying the heterogeneous use case.

Description

The Heterogeneous App is a test application that allows you to test the heterogeneous use case. It uses Capture + VISS + MSC + Display TIOVX.

Note
This app does not support heterogeneous camera on a single deserializer. Each camera module must be on a separate deserializer hub.

Supported plaforms

Platform Linux x86_64 Linux+RTOS mode QNX+RTOS mode SoC
Support NO YES YES J721e / J721S2 / J784S4 / J722S

Steps to run the application on J7 EVM (Linux + RTOS mode)

  1. Build the application and related libraries as mentioned in Build Instructions for Linux+RTOS mode
  2. Connect IMX728 on cam0 port and IMX390 on cam4 port of Fusion1/2
  3. Run the app as shown below
    cd /opt/vision_apps
    source ./vision_apps_init.sh
    ./run_app_heterogeneous.sh

Steps to run the application on J7 EVM (QNX + RTOS mode)

  1. Build the application and related libraries as mentioned in Build Instructions for QNX+RTOS mode
  2. Connect IMX728 on cam0 port and IMX390 on cam4 port of Fusion1/2
  3. Run the app as shown below
    cd /ti_fs/vision_apps
    . ./vision_apps_init.sh
    ./run_app_heterogeneous.sh
